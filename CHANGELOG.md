# Changelog

## [0.2.0] - 2026-06-22

### Исправлено
- Удален парсинг задач (VTODO) — теперь плагин работает только с событиями
- Исправлено определение ежедневных заметок (по началу имени)
- Улучшено отображение all-day событий — теперь показывается как "Весь день"

### Добавлено
- Поддержка all-day событий (VALUE=DATE)
- Новое поле `${timeDisplay}` в шаблоне
- Поддержка имен заметок с дополнительным текстом (например, `2026-06-22 - D22`)

## [0.1.1] - 2026-06-22

### Исправлено
- Добавлена поддержка парсинга задач (VTODO) из Яндекс.Календаря
- Добавлена поддержка вседневных событий (all-day events) с форматом `VALUE=DATE`
- Исправлена обработка `DTSTART` и `DTEND` без временной зоны
- Обновлен CalDAV запрос для включения задач (VTODO) в выборку

### Добавлено
- Новые поля в `CalendarEventDto`: `isTask`, `allDay`, `dueDate`, `priority`, `status`
- Возможность отображать задачи и события в едином списке
- Поддержка шаблонов с новыми переменными: `${isTask}`, `${allDay}`, `${dueDate}`, `${priority}`, `${status}`

## [0.0.1] - 2025-11-25

### Добавлено
- Первоначальный релиз плагина
- Базовая интеграция с Яндекс.Календарем через CalDAV
- Поддержка событий (VEVENT)
- Вставка событий в ежедневные заметки
- Настройка шаблона вывода событий
